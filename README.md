# Colegio Futuro Digital Final v3

SPA en Angular con:
- Login con JWT (mock)
- Guards
- Interceptor
- CRUD completo de usuarios y cursos
- Dashboard con tarjetas (counts)

## Pasos para ejecutar

1. Instalar dependencias frontend:

```bash
npm install
```

2. Instalar dependencias del mock API:

```bash
cd mock-api
npm install
cd ..
```

3. Levantar API:

```bash
npm run mock-api
```

4. Levantar Angular:

```bash
npm start
```

App: http://localhost:4200
API: http://localhost:3000
